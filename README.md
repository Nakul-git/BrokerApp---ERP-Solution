# BrokerApp---ERP-Solution
Multi-database ERP system for brokerage businesses with financial year–based sales tracking, billing, and reporting.

## 🚀 Overview
BrokerApp is a desktop/web-based ERP system designed for brokerage businesses to manage financial transactions, companies, and yearly sales efficiently using a multi-database architecture.

---

## 🧠 Core Architecture

### 1. Master Database
- User accounts & roles
- Company list
- System settings
- Financial year control

### 2. Company Database
- Company details (GST, address)
- Party/customer records
- Products/commodities
- Commission rules

### 3. Sales Database (Year-Based)
Example:
- sales2025
- sales2026

Includes:
- Sales & purchase entries
- Bills/invoices
- Ledger records
